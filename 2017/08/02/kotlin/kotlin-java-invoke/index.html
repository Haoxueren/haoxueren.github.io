二、一些特殊的调用规则

  1. 返回值为void的方法, 在 Kotlin 中调用时将返回 Unit。
  2. 符合 Java 的 Getter 和 Setter 规范的方法，在 Kotlin 中会被识别为属性。
  3. 某些 Kotlin 关键字在 Java 中是合法的标识符，如：in、object、is等，在Kotlin中使用时要使用反引号转义，如 `object`。
